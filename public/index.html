Seja Bem Vindo! Para usar corretamente esta API deve enviar a requisição pelo endpoint correto. Para obter todos os registros de conversões acesse: "localhost:8000/exchange"

Agora, se deseja efetuar uma conversão acesse "localhost:8000/exchange/{amount}/{from}/{to}/{rate}".

Abaixo tem uma demonstração:

Suponhamos que tenha 10 reais (BRL) e queira saber a conversão dela para o dólar (USD), sabendo que a taxa de converão está em 5.03, então escreve a URL desta maneira: localhost:8000/10/BRL/USD/5.03

Pode usar reais (BRL), dólar(USD) e o euro (EUR). Se escrever qualquer outra coisa sem ser isso, dará erro, assim como não escrevendo


OBS: Lembre-se de levantar o container antes de acessar os endpoints!
